# Doom Emacs Config

My [Doom Emacs](https://github.com/doomemacs/doomemacs) config.

## Installation

Clone this repo into `~/config/doom`:

```sh
git clone git@github.com:unindented/doomemacs-config.git ~/config/doom
```

Remember to sync your settings:

```
doom sync
```
